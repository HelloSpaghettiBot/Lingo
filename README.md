# 🌐 Discord Language Role Bot

A multilingual community bot that helps users pick their language roles using emoji reactions, automatically creates text/voice channels per language, and sets up translation commands with ease.

## 🚀 Features

- 🧠 Auto-creates all server categories, channels, and language roles
- 🗣️ Reaction-based language role assignment
- 🔐 Auto-manages channel permissions based on selected language
- 🌍 Sends translation setup commands to each language channel
- 🧹 `!clear` command to wipe messages from a channel (admin-only)
- 👋 Welcome messages that prompt new users to select their language
